GAME QUIZ TTS (TEKA - TEKI SANTAI)

- Di dalam quiz terdapat 5 soal di setiap levelnya
- Terdapat 3 macam level yaitu easy, mudah, dan gampang pada pilihan menu game nomor 2
- Jika game pertama kali di jalankan, maka level yang digunakan adalah level 1
- Di dalam menu game terdapat 3 pilihan, yaitu : 
  1. Main Game.
  2. Pilih Level.
  3. Keluar.  
- Jika menjawab 3 soal dengan jawaban yang salah, maka otomatis game akan kalah (Game Over)
- Jawaban yang salah tersebut nantinya akan ada Koreksi Jawaban
